CoderTrevor's Adventures in Deep Rock Galactic Modding

Join me as we learn to make blueprint mods for DRG together

These adventures aim to pick up where Buckminsterfullerene's excellent Blueprint Modding tutorial leaves off, so make sure you read that first!

  • Journey 0: Gotta go fast! - Make sure you can iterate on your mod quickly.
  • Journey 1: Where am I? - Print player position to the HUD and show your HUD in caves.
  • Journey 2: Let there be cube! - Spawn a new actor in the space rig.
  • Journey 3: I spy with my little eye... - See the name of (almost) any actor you're looking at. And delete whatever you want!
